**Message types (concrete `msg.type` strings):**

| type | payload | behavior |
|------|---------|----------|
| `theme:updateBook` | `{ centralQuestion, bookSynopsis }` partial | merge into pending doc, debounce save 300ms |
| `theme:updateBookCustom` | `{ key, value }` | set `bookCustomFields[key]=value`, debounce save |
| `theme:addTheme` | none | `addCentralThemeEntry` on pending doc, save immediately, re-render |
| `theme:removeTheme` | `{ themeId: string }` | filter `centralThemes`, confirm() in webview before postMessage |
| `theme:updateTheme` | `{ themeId, title?, summary?, notePath? }` | merge fields |
| `theme:toggleBookChapter` | `{ path: string }` | add/remove `path` in `bookLinkedChapters` |
| `theme:toggleThemeChapter` | `{ themeId, path: string }` | add/remove in that theme's `linkedChapters` |
| `thread:select` | `{ fileName: string }` | set selection, re-render |
| `thread:create` | none | host `showInputBox` for title (existing character pattern) |
| `thread:updateField` | `{ fileName, field, value }` | same semantics as character: name, title, body, custom:*, **not** touchesChapters via this message |
| `thread:setTouchesChapters` | `{ fileName, paths: string[] }` | replace `touchesChapters`, save immediately |
| `thread:addCustomField` | `{ fileName, fieldName }` | same as character |
| `thread:delete` | `{ fileName }` | confirm in webview, then delete |

**Themes tab UI (D-06):** Sections in order: Central question (textarea/input), Book synopsis (textarea), Book-level custom fields (repeat rows + "Add book field" button using `prompt()` for key), Book-level linked chapters (checkbox list from `chapterPickerOptions` bound to `bookLinkedChapters`), Central themes list (each: title, summary, note path input, linked chapters checkboxes for that theme's `linkedChapters`), buttons "Add central theme" → `theme:addTheme`.

**Threads tab:** Clone `renderCharactersTab` / `renderCharacterDetail` layout classes (use `thread-` CSS class prefix to avoid style collisions); list header button `+ New Thread` (`data-action="thread:create"`); detail: title, optional custom fields, **body** textarea, **Touches chapters** section: checkbox list over `chapterPickerOptions`, checked state from `profile.touchesChapters`; on change post `thread:setTouchesChapters`. Prominent label: "Touches chapters" (D-15). No search/filter (D-07).

**Tab bar:** `const TAB_IDS = ["outline", "cards", "themes", "characters", "places", "threads"]` and `TAB_LABELS.themes = "Themes"` (D-05).
