# LeanQuill

**VS Code extension for independent authors working in LeanPub-style markdown/git repositories.**

LeanQuill is an extension-first, local-first author workflow tool with optional companion views for dual-screen use. It organizes story intelligence, tracks editorial issues, and provides AI-assisted feedback while preserving the author's full ownership of all manuscript prose.

This folder contains portable, repository-agnostic planning artifacts. The project will move to a dedicated standalone repository before implementation begins.
